gitlab 搭建博客部署至远程

FM | Posted on 2020-03-11 | In 环境搭建 | 本文总阅读量 次

安装Hexo

需要 npm

提交至gitlab

需要 git

创建gitlab runner

进入 Setting > CI/CD Pipelines

  • 根据 How to setup a specific Runner for a new project 进行配置 gitlab-runner

    以 centos7 环境为例

    • 注册gitlab-runner

      若需要修改 gitlab-runner 注册信息, 只需进入 /etc/gitlab-runner/config.toml 进行编辑修改

页面出现可执行的runner即可, 在页面上也可对runner配置进行修改

配置完成后, 只需在 Project > CI configuration 进行配置即可触发 CI
.gitlab-ci.yml

若需要进行部署, 则在 .gitlab-ci.yml 中增加部署脚本内容即可
多任务模板:

配置Hexo部署

  • 部署至 git

    • npm install hexo-deployer-git --save
    • 在 _config.yml 增加配置内容

  • 部署至 sftp

    • npm install hexo-deployer-sftp --save
      • 在 _config.yml 增加配置内容

        Pipeline

  • 查看CI/CD 执行, 也可点击 Run Pipeline 进行手动触发

  • schedules 可进行定时执行

  • environment 可设置环境变量
